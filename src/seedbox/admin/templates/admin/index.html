Usage manual

  1. Define a cluster in Cluster tab.
  2. Add some nodes to cluster in Node tab. You will need at least one node with etcd server (three for fault tolerance), one k8s master (two for HA) and one k8s schedulable node. For testing purposes this can be a single node.
  3. Add a user with system:masters group and download generated kubeconfig.
  4. Configure your cluster servers for iPXE boot with this script: 
    #!ipxe
chain http://seedbox_host:seedbox_port/ipxe
  5. Startup the servers.
  6. After some time required to bootstrap full k8s cluster, you will be able to access it using kubectl and downloaded kubeconfig.
