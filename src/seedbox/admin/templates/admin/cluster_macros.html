  1. Download kubeconfig and activate it:
    $ export KUBECONFIG=/path/to/kubeconfig
  2. Install required role binding:
    $ kubectl create -f {{ request.url_root }}manifests/{{ model.name }}/kube-system-role-binding.yaml
  3. Init helm:
    $ helm init
  4. Install any addons:
    $ helm install -n RELEASE_NAME CHART_URL.
